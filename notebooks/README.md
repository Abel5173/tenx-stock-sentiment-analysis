# Jupyter Notebooks

This directory contains Jupyter notebooks used for data analysis, visualization, and model development.

## Notebook Organization

- `data_exploration/`: Notebooks for initial data analysis and visualization
- `model_development/`: Notebooks for developing and testing models
- `results/`: Notebooks for analyzing and presenting results

## Best Practices

1. Keep notebooks focused on a single task
2. Document your code with markdown cells
3. Use version control for your notebooks
4. Clear all outputs before committing
5. Use relative imports for project modules

## Running Notebooks

1. Activate the virtual environment
2. Start Jupyter:
```bash
jupyter notebook
```
3. Navigate to the desired notebook
